hdfscm
======

A custom ContentsManager_ for `Jupyter Notebooks`_ that stores contents on
HDFS.


Installation
------------

**Install with Pip:**

.. code::

    pip install jupyter-hdfscm

**Install from source:**

.. code::

    pip install git+https://github.com/jcrist/hdfscm.git
